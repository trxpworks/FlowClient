Why Flow

Premium CGUI Direction

Badlion-style cards, Lunar-style flow, readable contrast, search, favorites, settings pane, and central keybind manager.

Consumer Setup

First-run runtime setup, repair runtime, exact failed URL diagnostics, and safe fallback to existing Minecraft asset cache.

Safe Scope

Flow stays focused on HUD, visual, performance, cosmetics, and QoL features. No hidden autoclickers, packet abuse, or combat automation.

FAQ

Before installing

Does Flow include OptiFine?

No. OptiFine is paused for now and Flow does not redistribute OptiFine.

What Minecraft version?

Flow is currently focused on Minecraft 1.8.9.

What if setup fails?

Use Repair Runtime and Export Diagnostics in the launcher so the exact failed URL and runtime status can be reviewed.
